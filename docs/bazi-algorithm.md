# 八字 AI 解读算法系统

## 概述

本系统为 DeepSeek 提供**结构化、标准化的八字分析数据**，确保每次解读基于统一规则，避免随机性。

---

## 核心算法模块

### 1. 日主强弱分析 (`calculateRiZhuStrength`)

**计算维度：**
- **得令**（季节）：月令五行与日主关系 (+20~25分)
- **得地**（根气）：日支、时支、年支藏干有日主五行 (+5~15分)
- **得势**（帮扶）：天干有比劫、印星 (+10~12分)

**判定标准：**
- 总分 ≥ 70：身强（需要克泄耗）
- 40 ≤ 总分 < 70：中和
- 总分 < 40：身弱（需要生扶）

---

### 2. 五行旺衰分析 (`calculateWuXingBalance`)

**力量计算：**
- 天干：+1分
- 地支：+1分
- 藏干主气：+0.6分
- 藏干中气：+0.3分
- 藏干余气：+0.1分

**状态判定：**
- ≥25%：旺
- 20%~25%：相
- 15%~20%：平
- 10%~15%：弱
- <10%：死

---

### 3. 十神配置分析 (`calculateShiShenConfig`)

**输出：**
- 各十神力量分布
- 主导十神（最旺的2-3个）
- 缺失十神
- 性格特征总结
- 适合职业类型
- 求财方式

---

### 4. 格局判定 (`calculateGeJu`)

**特殊格局（从格）：**
- 从财格：日主极弱 + 财星极旺
- 从杀格：日主极弱 + 官杀极旺
- 从儿格：日主极弱 + 食伤极旺
- 从强格：日主极强 + 比劫极旺

**普通格局：**
- 正官格、七杀格、正印格、偏印格
- 正财格、偏财格、食神格、伤官格
- 建禄格、月刃格

---

### 5. 用神喜忌分析 (`calculateYongShen`)

**身强时（需要克泄耗）：**
- 官杀旺 → 用神：食伤（制官杀）
- 印星旺 → 用神：财星（破印）
- 比劫旺 → 用神：官杀（制比劫）
- 食伤旺 → 用神：财星（泄食伤）

**身弱时（需要生扶）：**
- 官杀旺 → 用神：印星（化杀生身）
- 财星旺 → 用神：比劫（帮身抗财）
- 食伤旺 → 用神：印星（制食伤）
- 一般身弱 → 用神：印星（生身）

---

### 6. 运势评分算法 (`calculateBaziScore`)

**计算公式：**
- 事业分 = 官杀力量×30% + 印星力量×20% + 格局纯正度×20% + 大运配合×30%
- 财运分 = 财星力量×35% + 食伤力量×25% + 身财平衡×20% + 大运配合×20%
- 感情分 = 配偶星力量×40% + 桃花/红鸾×15% + 日支稳定度×25% + 大运配合×20%
- 健康分 = 五行平衡度×30% + 日主强弱适中×25% + 无严重冲克×25% + 大运配合×20%
- 综合分 = 事业×30% + 财运×25% + 感情×25% + 健康×20%

## DeepSeek 提示词规范

### System Prompt 规则

```
【重要规则】
1. 必须基于提供的算法分析数据进行解读，不得随意编造
2. 日主强弱、五行旺衰、格局判断等核心结论必须与算法数据一致
3. 运势评分直接使用算法计算结果，不得擅自修改
4. 用神喜忌必须遵循算法推导逻辑
5. 语言专业但易懂，避免过于晦涩的术语
6. 给出具体、实用的建议，结合现代生活场景
7. 保持客观理性，避免绝对化的断言

【解读风格】
- 先陈述算法分析结果（事实）
- 再给出命理解读（分析）
- 最后提供实用建议（应用）
```

### Temperature 设置

- `temperature: 0.5` - 降低随机性，更遵循规则

## API 响应格式

```typescript
{
  success: true,
  analysis: string,        // DeepSeek 生成的 HTML 格式解读
  algorithmData: {         // 算法分析数据（供前端使用）
    riZhu: RiZhuStrength,
    wuXing: WuXingBalance,
    shiShen: ShiShenConfig,
    geJu: GeJuAnalysis,
    yongShen: YongShenAnalysis,
    scores: BaziScore,
    tags: string[]
  },
  model: string,
  usage: object
}
```

## 使用示例

```typescript
import { analyzeBaziComplete } from '@/app/lib/bazi';

// 获取八字详情
const detail = getBaziDetail(1995, 12, 25, 12, 'male');

// 完整分析
const analysis = analyzeBaziComplete(detail, currentDaYun);

// 使用分析结果
console.log(analysis.riZhu.strength);  // 'strong' | 'medium' | 'weak'
console.log(analysis.scores.overall);  // 综合评分
console.log(analysis.tags);            // 特征标签
```
